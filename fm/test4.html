    1. Liquid pouring in/leaking out: units are cm3s-1 which is volume divided by time, so: \(\dfrac{\mathrm{d}V}{\mathrm{d}t} = 1600 - k_1\sqrt{h}\)
      Volume of cylinder: \(V = Ah \Rightarrow A = \dfrac{V}{h}\) where \(A\) is the cross sectional area. If cross sectional area is constant, as volume changes height must change \(A = \dfrac{\mathrm{d}V}{\mathrm{d}h}\)
      Put these together: \(\dfrac{\mathrm{d}h}{\mathrm{d}t} = \dfrac{\mathrm{d}h}{\mathrm{d}V} \times \dfrac{\mathrm{d}V}{\mathrm{d}t} = \dfrac{1}{A}(1600 - k_1\sqrt{h})\)
      Given \(A = 4000\): \(\dfrac{\mathrm{d}h}{\mathrm{d}t} = \dfrac{1600}{4000} - \dfrac{k_1}{4000}\sqrt{h} = 0.4 - k\sqrt{h}\) where \(k = k_1/4000\)

    2. \(\dfrac{\mathrm{d}V}{\mathrm{d}t} = 1600 - k_1\sqrt{h}\)
      Leaking out is the second part, so \(k_1\sqrt{h} = 400\)
      When \(h=25\): \(k_1\sqrt{25} = 400 \Rightarrow k_1 = 80\)
      \(k = \dfrac{k_1}{4000} = 0.02\)

    3. \(\displaystyle\dfrac{\mathrm{d}h}{\mathrm{d}t} = 0.4 - 0.02\sqrt{h} \Rightarrow \int\dfrac{1}{0.4-0.02\sqrt{h}}\ \mathrm{d}h = \int 1\ \mathrm{d}t\)
      Multiply left hand side's top and bottom by 50: \(\displaystyle\int\dfrac{50}{20-\sqrt{h}}\ \mathrm{d}h = \int 1\ \mathrm{d}t\)
      Height is from 0 to 100: \(\displaystyle\int_0^{100}\dfrac{50}{20-\sqrt{h}}\ \mathrm{d}h = t\)

    4. \(h = (20-x)^2 \Rightarrow \sqrt{h} = 20 - x \Rightarrow 20 - \sqrt{h} = x\)
      \(\dfrac{\mathrm{d}h}{\mathrm{d}x} = -2(20 - x) = 2x - 40 \Rightarrow \mathrm{d}h = (2x-40)\ \mathrm{d}x\)
      \(\displaystyle\int\dfrac{50}{20-\sqrt{h}}\ \mathrm{d}h = \int \dfrac{50}{x}(2x-40)\ \mathrm{d}x = \int 100 - \dfrac{2000}{x}\ \mathrm{d}x = 100[x - 20\ln x]\)
      Lower limit: \(h = 0 \Rightarrow x = 20\), upper limit: \(h = 100 \Rightarrow x = 10\)
      Substitute these in: \(100\Big[(10 - 20\ln 10) - (20 - 20\ln 20)\Big] = 100\Big[-10 - 20\ln 10 + 20\ln 20)\Big]\)
      Use log rule: \(= 100\Big[-10 + 20\ln \dfrac{20}{10}\Big] = 100\Big[20\ln 2 - 10\Big] = 2000\ln 2 - 1000\)

      \(\displaystyle\int\dfrac{\mathrm{d}x}{(1-x^2)^{\frac{3}{2}}} = \int \dfrac{\cos\theta}{(1 - \sin^2\theta)^{\frac{3}{2}}}\ \mathrm{d}\theta = \int \dfrac{\cos\theta}{(\cos^2\theta)^{\frac{3}{2}}}\ \mathrm{d}\theta = \int \dfrac{\cos\theta}{\cos^3\theta}\ \mathrm{d}\theta = \int \sec^2\theta\ \mathrm{d}\theta = [\tan\theta]\)
      Lower limit: \(x = 0 \Rightarrow \theta = 0\), upper limit: \(x = 0.5 \Rightarrow \theta = \dfrac{\pi}{6}\)
      Substitute in: \(\tan\dfrac{\pi}{6} - \tan 0 = \dfrac{1}{\sqrt{3}}\)

    2. \(\displaystyle\int\theta f(\theta)\ \mathrm{d}\theta = \dfrac{1}{2}\int\theta + 7\theta\cos2\theta\ \mathrm{d}\theta\)
      Integrate \(\theta\cos2\theta\) by parts using \(u = \theta\) and \(v' = \cos2\theta\) to get \(u' = 1\) and \(v = \dfrac{1}{2}\sin2\theta\)
      Put these together: \(\displaystyle\int\theta\cos2\theta = \dfrac{1}{2}\theta\sin2\theta + \dfrac{1}{4}\cos2\theta\)
      \(\displaystyle\dfrac{1}{2}\int\theta + 7\theta\cos2\theta\ \mathrm{d}\theta = \dfrac{1}{2}\Big[\theta^2 + 7\left(\dfrac{1}{2}\theta\sin2\theta + \dfrac{1}{4}\cos2\theta\right)\Big] = \dfrac{\theta^2}{4} + \dfrac{7\theta}{4}\sin2\theta + \dfrac{7}{8}\cos2\theta\)
      Lower limit: 0, upper limit: \(\pi/2\)
      Put in limits: \(\left(\dfrac{\pi^2}{16}+0-\dfrac{7}{8}\right) - \left(0 + 0 + \dfrac{7}{8}\right) = \dfrac{pi^2}{16} - \dfrac{7}{4}\)
