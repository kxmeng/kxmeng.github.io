Quartics

  1. \(w = 1 - i,\ w^2 = -2i,\ w^3 = -2-2i,\ w^4 = -4\)
  2. \(w^4 + 3w^3 + pw^2 + qw + 8 = 0\)
    \(-4 + 3(-2-2i) + p(-2i) + q(1-i) + 8 = 0\)
    Real: \(-4 - 6 + q + 8 = 0 \Rightarrow q = 2\)
    Imaginary: \(-6i -2pi -qi = 0 \Rightarrow p = -4\)
  3. Let \(\alpha,\ \beta,\ \gamma,\ \delta\) be the roots. \(z^4 + 3z^3 + pz^2 + qz + 8 = (z-\alpha)(z-\beta)(z-\gamma)(z-\delta)\)
    One root is \(1 - i\), so we can find another root.
    Once two roots are found, use polynomial division to turn the quartic into a quadratic (do not do division with complex numbers)
