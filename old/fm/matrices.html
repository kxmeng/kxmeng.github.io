  • 2.4 Q12
    \(\textbf{D}^2 = \begin{pmatrix}a^2-b^2 & -2ab\\2ab & a^2-b^2\end{pmatrix}\)
    Comparing the matrices: \(a^2-b^2 = \dfrac{1}{\sqrt{2}} \Rightarrow a^2 = \dfrac{\sqrt{2}}{2} + b^2\) and \(2ab = \dfrac{1}{\sqrt{2}} \Rightarrow a^2b^2 = \dfrac{1}{8} \Rightarrow a^2 = \dfrac{1}{8b^2}\)
    \(\dfrac{\sqrt{2}}{2} + b^2 = \dfrac{1}{8b^2} \Rightarrow 8b^4 + 4\sqrt{2}b^2 - 1 = 0\)
    Quadratic formula: \(b^2 = \dfrac{-4\sqrt{2}\pm\sqrt{32 + 32}}{16} = \dfrac{-4\sqrt{2}+8}{16} = \dfrac{2-\sqrt{2}}{4}\) (\(b\) is a real number so \(b^2 > 0\)) and \(a^2 = \dfrac{4}{8(2-\sqrt{2})} = \dfrac{4(2+\sqrt{2})}{8(4-2)} = \dfrac{2+\sqrt{2}}{4}\)
    \(a = \dfrac{\sqrt{2+\sqrt{2}}}{2},\ b = \dfrac{\sqrt{2-\sqrt{2}}}{2}\)
    Note that you can rationalise the denominator in the solution for \(b\) given in the textbook.
  • 2.5 Q7
    If line \(y=mx\) is invariant, every point that satisfies this equation must be unchanged by the transformation. These points have coordinates \((x, y) = (x, mx)\), and we can apply the transformation to these points:
    \(\begin{pmatrix}\dfrac{1-m^2}{1+m^2}&&\dfrac{2m}{1+m^2}\\\dfrac{2m}{1+m^2}&&\dfrac{m^2-1}{1+m^2}\end{pmatrix}\renewcommand\arraystretch{2}\begin{pmatrix}\color{white}\dfrac{1^2}{1^2}\color{black}x\color{white}\dfrac{1^2}{1^2}\color{black}\\\color{white}\dfrac{1^2}{1^2}\color{black}mx\color{white}\dfrac{1^2}{1^2}\color{black}\end{pmatrix} = \begin{pmatrix}\dfrac{x(1-m^2)+2m^2x}{1+m^2}\\\dfrac{2mx + mx(m^2-1)}{1+m^2}\end{pmatrix} = \begin{pmatrix}\dfrac{x(1-m^2+2m^2)}{1+m^2}\\\dfrac{mx(2 + m^2-1)}{1+m^2}\end{pmatrix} = \begin{pmatrix}\dfrac{x(1+m^2)}{1+m^2}\\\dfrac{mx(1 + m^2)}{1+m^2}\end{pmatrix}\)
  • 6.2 Q8
    Singular, so \(|M| = 0 = ad - bc \Rightarrow \color{red}ad = bc\)
    \(\begin{pmatrix}a&b\\c&d\end{pmatrix}^2 = \begin{pmatrix}aa+\color{red}bc&ab+bd\\ac+cd&\color{red}bc\color{black}+dd\end{pmatrix} = \begin{pmatrix}aa+\color{red}ad&ab+bd\\ac+cd&\color{red}ad\color{black}+dd\end{pmatrix} = \begin{pmatrix}a(a+d)&b(a+d)\\c(a+d)&d(a+d)\end{pmatrix} = (a+d)M\)
    \(M^3 = \color{green}M^2\color{black}M = \color{green}(a+d)M\color{black}M = (a+d)M^2 = (a+d)(a+d)M = (a+d)^2M\)
    An extra multiplication by \(M\) increases the power of the scalar factor by 1, so \(M^n = (a+d)^{n-1}M\)
