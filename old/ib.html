    The first terms of an arithmetic sequence are \(\dfrac{1}{\log_2x},\dfrac{1}{\log_8x},\dfrac{1}{\log_{32}x},\dfrac{1}{\log_{128}x},...\)

    Find \(x\) if the sum of the first \(20\) terms of the sequence is equal to \(100\).

    [7 marks]

    A geometric sequence has first term \(a\), common ratio \(r\) and sum to infinity \(76\).

    A second geometric sequence has first term \(a\), common ratio \(r^3\) and sum to infinity \(36\).

    Find \(r\).

    Use the sum to infinity formula to get \(\dfrac{a}{1-r} = 76\) and \(\dfrac{a}{1-r^3} = 36\)

    These equations can be solved simultaneously e.g. by elimination of \(a\) to get an equation in \(r\): \(76(1-r) = 36(1-r^3)\)

    Expanding and rearranging gives a cubic which can be solved by recognising that \((r-1)\) is a factor, but it is easier to divide by \((1-r)\) to give \(\dfrac{76}{36} = \dfrac{1-r^3}{1-r} = r^2+r+1\)

    Simplifying this this gives \(r^r+r - \dfrac{10}{9} = 0\)

    Solving this e.g. by using the formula gives \(r=\dfrac{12}{18}, -\dfrac{30}{18}\)

    This gives \(r=\dfrac{2}{3}\) as the only solution since the sum to infinity only exists for \(-1 < r < 1\).

    3 boys and 3 girls are to sit on a bench for a photograph.

    a) Find the number of ways this can be done if the three girls must sit together.

    b) Find the number of ways this can be done if the three girls must all sit apart.

    a) Take the 3 girls as a single "person". Number of ways of arranging 4 people is \(4!\).

    Number of ways of arranging the 3 girls within themselves is \(3!\). Total number of ways is \(4!\times3! = 144\)

    b) Number of ways of arranging 3 girls within themselves is \(3!\). This is the same for the boys.

  • 1.4

    • Proof by mathematical induction.
    • Links to a wide variety of topics, for example, complex numbers, differentiation, sums of series and divisibility.
    [7 marks]

    Use the method of mathematical induction to prove that \(5^{2n} -24n - 1\) is divisible by \(576\) for \(n\in\mathbb{Z}^+\)

    Test case: when \(n=1\), \(5^2 - 24 - 1 = 0\), which is divisible by \(576\) so this is true.

    Assume true when \(n=k\) for some number \(k\): that \(5^{2k} - 24k - 1 = 576N\) where N is some number.

    Consider \(n=k+1\): \(5^{2(k+1)} - 24(k+1) - 1 = 25\times 5^{2k} - 24k - 25\)

    From the assumption, \(5^{2k} = 576N + 24k + 1\)

    Substituting this in gives \(25\times(576N + 24k + 1) - 24k - 25\)

    Expanding this out gives \(576k + 576N\times 25 = 576(k+25N)\) which is a multiple of 576

    Proof: If \(n=k\) is true, then true for \(n = k+1\). True for \(n=1\) so true for all \(n\in\mathbb{Z}^+\)

    [7 marks]

    Consider the complex number \(z=\cos\theta + \mathrm{i}\sin\theta\)

    a) Use De Moivre's theorem to show that \(z^n+z^{-n} = 2\cos n\theta,\ n\in\mathbb{Z}^+\)

    b) Expand \(\left(z+z^{-1}\right)^4\)

    c) Hence show that \(\cos^4\theta = p\cos4\theta + q\cos2\theta + r\), where \(p\), \(q\) and \(r\) are constants to be determined.

    a) \(z^n = \cos n\theta + \mathrm{i}\sin n\theta\), \(z^{-n} = \cos -n\theta + \mathrm{i}\sin -n\theta\)

    By symmetry, \(\cos -n\theta = \cos n\theta\) and \(\sin -n\theta = -\sin n\theta\)

    \(z^n + z^{-n} = \cos n\theta + \cos n\theta + \mathrm{i}\sin n\theta - \mathrm{i}\sin n\theta\)

    \(=2\cos n\theta\)

    b) Expand the brackets to give \(\left(z+z^{-1}\right)^4 = z^4 + 4z^3z^{-1} + 6z^2z^{-2} + 4zz^{-3}+z^{-4}\)

    c) Using part a, \(z + z^{-1} = 2\cos\theta\)

    Using part b, \((2\cos\theta)^4 = z^4 + 4z^3z^{-1} + 6z^2z^{-2} + 4zz^{-3}+z^{-4}\)

    Simplifying this gives \(z^4 + 4z^2 + 6 + 4z^{-2} + z^{-4}\)

    Rearranging gives \(\left(z^4 + z^{-4}\right) + 4\left(z^2+z^{-2}\right)+6\)

    Using part a again, \(z^4+z^{-4} = 2\cos4\theta\) and \(z^2+z^{-2} = 2\cos2\theta\)

    Finally, \((2\cos\theta)^4 = 16\cos^4\theta = 2\cos4\theta + 8\cos2\theta + 6\)

    \(\cos^4\theta = \dfrac{1}{8}\cos4\theta + \dfrac{1}{2}\cos2\theta + \dfrac{3}{8}\)

    • Conjugate roots of polynomial equations with real coefficients.
    • Link to 2.5 and 2.7.
    [7 marks]

    A polynomial \(p(x)\) with real coefficients is of degree five. The equation \(p(x) = 0\) has a complex root \(2+\mathrm{i}\). The graph of \(y = p(x)\) has the x-axis as a tangent at \((2, 0)\) and intersects the coordinate axes at \((−1, 0)\) and \((0, 4)\).

    Find \(p(x)\) in factorised form with real coefficients.

    Complex roots always come in conjugate pairs, so \(2-\mathrm{i}\) must also be a root.

    Intersects the x-axis at -1 which gives a third root, and touches the x-axis at 2 which gives a double root at 2.

    Using the 5 roots, the polynomial must be \(a(x-2-\mathrm{i})(x-2+\mathrm{i})(x+1)(x-2)(x-2)\)

    Multiplying out the complex conjugate pair gives \(a(x^2-4x+5)(x+1)(x-2)(x-2)\)

    This goes through the coordinates \(0,4)\) so \(a(5)(1)(-2)(-2)=4 \Rightarrow a=\dfrac{1}{5}\)
